1 简单描述Zabbix具有哪些监控功能

参考答案

  • 具备常见的商业监控软件所具备的功能
  • 主机性能监控、网络设备监控、数据库监控等
  • 支持多种报警机制
  • 支持自动发现网络设备和服务器
  • 可以通过配置自动发现服务器规则来实现
  • 支持分布式，能集中展示、管理分布式的监控点
  • 编写插件容易，可以自定义监控项
  • 具有实时绘图功能

2 简述源码编译安装Zabbix平台的操作步骤

参考答案

copytextpop-up
  1. [root@zabbixserver ~]# tar  zabbix-2.2.1.tar.gz -C /usr/src
  2. [root@zabbixserver ~]# cd /usr/src/zabbix-2.2.1/
  3. [root@zabbixserver zabbix-2.2.1]# ./configure \
  4. >--prefix=/usr/local/zabbix --enable-server --enable-proxy \
  5. >--enable-agent --with-mysql=/usr/bin/mysql_config \
  6. > --with-net-snmp --with-libcurl
  7. [root@zabbixserver zabbix-2.2.1]# make && make install
