1 简述什么是LNMP

参考答案

LNMP是网站运行平台。

  • L：操作系统（Linux）
  • N：网站服务软件（Nginx）
  • M：数据库服务软件（MariaDB）
  • P：网站开发语言（PHP、perl、python）

3 地址重写的好处

参考答案

  • 缩短URL，隐藏实际路径提高安全性；
  • 易于用户记忆和键入；
  • 易于被搜索引擎收录。

4 使用Nginx实现域名跳转

要求：访问www.tarena.com时自动跳转至bbs.tarena.com

参考答案

copytextpop-up
  1. [root@nginx ~]# cat /usr/local/nginx/conf/nginx.conf
  2. .. ..
  3. server {
  4.         listen       80;
  5.         server_name  www.tarena.com;
  6. location / {
  7.     root   html;
  8. index  index.html index.htm;
  9. rewrite ^/ http://bbs.tarena.com/;
  10. }
  11. }
[root@nginx ~]# cat /usr/local/nginx/conf/nginx.conf
